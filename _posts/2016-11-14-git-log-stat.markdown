## git log

查看所有提交记录


## git log -n 1

查看最近一次提交

## git log -n 1 --stat

查看最近一次提交的所有文件

## git log -n 3 --stat

查看最近三次提交的所有文件

## git log -n 1 -p

查看最近提交的所有细节
