年底了，总结一下最近一年研究过的开源项目：

```vim
├── AlgorithmVisualization                          # 用于算法教学的算法可视化项目，包括机器学习算法
├── DeepBeautyPredictor                             # 2015 年 ACM 深度人脸美化的 Tensorflow 实现
├── HanLP                                           # 中文分词的试验场
├── PDFlib-Lite-7.0.5p3                             # pdf 的开源 C lib
├── PoemRNN                                         # RNN 自动生成古诗
├── cardocr                                         # 卡片 OCR 识别
├── cmus-lrc                                        # 我自己的 cmus 歌词插件
├── cuma                                            # 个人兴趣项目使用 curses 写的文件浏览器
├── deepdive                                        # 斯坦福的新的数据管理系统，可从更少的结构化数据和统计推断中提取结构化知识
├── dgjqxc                                          # （这是个什么鬼？？）
├── disease_etiology_model                          # 疾病病例推断模型，疾病关联
├── face-analysis-sdk                               # 人脸解析
├── graph-extract                                   # neu4j 和 tinkerpop 的图数据转换及解析
├── heat-transfer                                   # 热传导方程式求解
├── imageboundary                                   # 图像边界检测
├── itchat                                          # 腾讯微信机器人接口（基于网页版，但腾讯即将关闭该服务）
├── kugod                                           # 自己的酷狗下载器
├── m3u8                                            # 自己的 m3u8 视频文件下载
├── mobikecup                                       # 摩拜 cup 竞赛（没拿到名次）
├── openface                                        # 开源里最好用的人脸识别项目
├── pptx                                            # ppt 的读写库
├── pyharu                                          # libharu(读写pdf的开源库，解决阿拉伯文的右排问题)的python封装
├── remove_git_big_file                             # 彻底铲除 git 仓库大文件的利器 bfg.jar
├── snorkel                                         # 斯坦福的机器学习给数据自动打标签的库
├── tensorflow                                      # google 的 tf 不必多说
├── tensorflow_cookbook                             # 学习的最快方式就是看 cookbook (前提是有初步了解)
├── tootstream                                      # mastosh(长毛象，类似 twitter) 的终端
├── tspredict                                       # 自己的时间序列 testful 算法封装
├── wechat-spider                                   # 微信公众号爬虫
├── wxbotmanager                                    # 多微信号管理的项目
├── zanimport                                       # 有赞、淘宝、京东的数据导出与导入脚本
└── zore                                            # 中文信息语义抽取项目（湾湾写的）
```
